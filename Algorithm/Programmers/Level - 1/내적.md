# 내적

# 문제 설명

1. 길이가 같은 두 1차원 정수 배열 a, b가 매개변수로 주어진다.

2. a와 b의 내적을 return하라 a와 b의 내적은 

```

a[0]*b[0] + a[1]*b[1]..

```

# 제한 사항

1. a와 b의 길이는 1이상 1,000 이하

2. a,b의 모든 수는 -1,000 ~ 1,000 이하

# 해결 방안

1. 해당 값 곱하고 해당 값 더하고..

# 풀이

- 나의 풀이

```

for ( int i = 0; i<a.length; i++) {
        answer+= a[i]*b[i];
}

```
