# 수박수박수박수박수박수?

# 문제 설명

1. 길이가 n이고, "수박수박수박수...."와 같은 패턴을 유지하는 문자열을 리턴하는 함수, solution을 완성하세요. 

2. 예를들어 n이 4이면 "수박수박"을 리턴하고 3이라면 "수박수"

# 제한 사항

1. n은 길이 10,000

# 해결 방안

1. 홀수면 수, 짝수면 박

# 풀이

- 나의 풀이

```

 String answer = "";
        
for (int i = 1 ;i <= n ; i++) 
{
    answer += i%2 == 1 ? "수" : "박";
}

return answer;

```
