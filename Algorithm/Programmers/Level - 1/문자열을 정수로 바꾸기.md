# 문자열을 정수로 바꾸기

# 문제 설명

1. 문자열 s를 숫자로 변환한 결과를 반환하는 함수

# 제한 사항

1. s의 길이는 1 ~ 5

2. s의 맨 앞에는 +, -가 올 수 있음

3. s는 부호와 숫자로만 이루어져있다.

4. s는 0으로 시작하지 않음

5. str이 1234이면 1234를 반환 str이 -1234이면 -1234를 반환

# 해결 방안

1. int answer = Integer.parseInt(s);

# 풀이

- 나의 풀이

```

int answer = Integer.parseInt(s);

```
